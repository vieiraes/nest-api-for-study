# Api simples para cadastro de usuarios em NEST

Desenvolvido para aprendizagem na plataforma.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
